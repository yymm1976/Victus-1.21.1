# Victus

A Minecraft mod for **NeoForge 1.21.1** that adds a custom heart aspects system.

## Features

- **Heart Aspects**: Replace your default hearts with special aspect hearts, each granting unique abilities
- **20+ Heart Types**: Including Blazing, Icy, Diamond, Emerald, Golden, Iron, Creeper, Totem, and more
- **Broken Heart Mechanic**: Hearts can break under damage, entering a cooldown before recharging
- **Potion Aspect**: Store and apply potion effects when the heart breaks
- **Victus Journal**: In-game guide to discover and craft heart aspects

## Building

```bash
./gradlew build
```

The output JAR will be in `build/libs/`.

## Installation

1. Install [Minecraft 1.21.1](https://www.minecraft.net/) with [NeoForge 21.1.x](https://neoforged.net/)
2. Place the mod JAR into your `.minecraft/mods/` folder
